Problem B：数字操作

题意分析

给定一个数，对其可以进行任意次数的操作

操作定义为：

  • 将整数 n 乘以任意一个正整数 x
  • 将整数 n 替换为 √n（执行此操作的前提是 √n 为整数）。

分解质因数：

例：n = 5184

5184 = 2^6^ + 3 ^4^

每次进行开平方操作我们都需要将指数 / 2，所以我们的目的就是在进行times = 0 或 1次乘法操作后，整数n可以一直开平方到两质因数之积

  • 先对质因数进行分解，ms存储质因数，qs存储指数
  • 取指数最大值，同时寻找一个2的次方数m >= max(qs)
  • 根据以上计算出需要乘的数news
    • 若news为1，则不需要乘积，直接返回sqrt(m)
    • 否则返回sqrt(m) + 1
